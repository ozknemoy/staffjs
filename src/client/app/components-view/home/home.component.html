Чтобы разрешить открывать файлы docx скачанные браузером необходимо снять настройку в ворде:
Файл > Параметры > Центр управления безопасностью > Параметры центра управления безопасностью. > Защищенный просмотра > Снемите первую галочку "Включить защищенный просмотр для файлов из Интернета".
Для английской версии file > options > trust center > кнопка trust center settings > protected view > снемите первую галочку (enable protectd view for files ... from internet)
